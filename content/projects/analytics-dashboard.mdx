## Overview

Live event streams with anomaly detection, alerting, and time window analysis.

## Highlights

- WebSockets stream ~5k events/sec to the UI
- Downsampling server-side for smooth charts
- Role-based access and saved views for teams

## Outcomes

- 99.9% uptime over last quarter
- Median chart render < 60ms on commodity devices
